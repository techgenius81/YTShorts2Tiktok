# YTShorts2TikTok

YTShorts2TikTok is a public tool created by me (@techgenius81) that lets you provide a YouTube Shorts URL, automatically fetches the video, and uploads it to your TikTok account with either a custom caption or the video's default caption.

## How to Use

1. Install **[EditThisCookie V3](https://chromewebstore.google.com/detail/editthiscookie-v3/ojfebgpkimhlhcblbalbfjblapadhbol)**.
2. Go to **https://www.tiktok.com/** and log into your TikTok account.
3. Open the **EditThisCookie V3** extension and click the **Download** button at the top.
4. ![Button Preview](https://i.imgur.com/rXITtVd.png)
5. Open the downloaded cookie file, copy all of its contents, and paste them into `cookies.json` in the project folder.
6. Install **Python 3.10+**.
7. Run `install.bat` to install all required dependencies.
8. Once installation is complete, run `run.bat`.
9. Paste the YouTube Shorts URL into the tool.
10. Optionally enter a custom caption. If left blank, the original YouTube caption will be used.
11. Click **Post**. The tool will automatically download the YouTube Short and upload it to your TikTok account.
